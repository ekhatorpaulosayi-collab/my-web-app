🧪 Test Customer Chat Visibility

📝 Instructions

Step 1: Make sure you've run the SQL migration in Supabase
Step 2: Open this in an incognito/private browser window
Step 3: Visit your store and send a test chat message

🔗 Quick Links

Open these in an incognito/private window:

  • Test Store (if you have one)
  • Your Main App

After sending a test chat, check here:

  • Customer Chats Page

✅ Verification Checklist

🎯 Expected Result

After completing all steps, you should see:
  • Your test conversation in the Customer Chats page
  • Message count and timestamps
  • Visitor session ID or name (if provided)
  • Active/Recent/Inactive status indicators

❓ Troubleshooting

If you don't see conversations:
  • Make sure you used an incognito/private window for testing
  • Check that the SQL migration ran successfully (no red errors)
  • Try refreshing the Customer Chats page
  • New conversations only - old ones won't appear
If SQL migration failed:
  • Try running each section of the SQL separately
  • Check for any specific error messages
  • Some "already exists" errors are okay
