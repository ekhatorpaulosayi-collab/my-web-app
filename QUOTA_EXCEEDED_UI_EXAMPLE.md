## What Happens When They Click "Upgrade Now":

1. **Redirects to**: `/subscription` page
2. **Shows**: All available plans with current plan highlighted
3. **Process**: Select new plan → Pay with Paystack → Instant activation
4. **Result**: Returns to chat with new limit active immediately

## Visual Features:

### Before Limit:
- ✅ Green chat bubble
- ✅ Instant AI responses
- ✅ Counter shows remaining chats

### At Limit:
- 🔴 Message appears instead of AI response
- 🔴 Clear explanation of limit reached
- 🔴 Pricing for upgrades shown
- 🔴 One-click upgrade button

### After Upgrade:
- ✅ Immediately continues conversation
- ✅ New limit shown in header
- ✅ No page reload needed

## Mobile View:
Same message appears but formatted for mobile:
- Full-width card
- Larger upgrade button
- Touch-friendly interface

## Key Points:

1. **Non-Intrusive**: Appears only when they actually try to send a message
2. **Informative**: Shows exact usage (30/30) not just "limit reached"
3. **Actionable**: One-click upgrade right there
4. **Smart**: Shows only higher tiers, not lower ones
5. **Persistent**: Message stays in chat history as reminder
